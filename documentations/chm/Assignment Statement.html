Assignment Statement:

Assignment statement assigns a value from a variable or function or data value to another variable. This is done by using the '='

 

Examples 1:

myname = "Jack"

myage = "25"

 

here the value is assigned to the variable 'myname' and 'myage'. With LSL variables being data type agnostic and no definition required, any data type could be assigned to the variable.

 

Example 2:

myname = "Jack"

name = myname

 

here - using the '=' data from 'myname' is assigned to the variable 'name'

 

Example 3:

myage = randomstring()

 

here the function randomnumeber() returns a numeric data type which is assigned to the variable 'myage'
